# 소개

이 문서 사이트는 Nextra를 사용하여 만들어졌습니다.

## Nextra란?

Nextra는 Next.js 기반의 정적 사이트 생성기로, 특히 문서 사이트 제작에 최적화되어 있습니다.

### 주요 특징

- **마크다운 기반**: MDX를 사용하여 React 컴포넌트와 마크다운을 함께 사용
- **파일 시스템 라우팅**: 파일 구조가 곧 URL 구조
- **기본 제공 컴포넌트**: Callout, Tabs, Steps 등 문서에 유용한 컴포넌트
- **검색 기능**: 내장된 전체 텍스트 검색
- **i18n 지원**: 다국어 문서 사이트 구축 가능

## 이 사이트에 대해

이 문서 사이트는 다음과 같은 목적으로 만들어졌습니다:

1. **지식 공유**: 팀 내 지식과 경험을 체계적으로 정리
2. **온보딩**: 새로운 팀원이 빠르게 적응할 수 있도록 도움
3. **레퍼런스**: API와 가이드를 한 곳에서 확인
