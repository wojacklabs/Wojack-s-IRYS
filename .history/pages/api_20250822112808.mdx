# API 레퍼런스

API 문서를 체계적으로 작성하는 방법을 알아봅니다.

## API 문서 작성 예시

### 함수 문서화

```typescript
/**
 * 두 숫자를 더합니다.
 * @param a - 첫 번째 숫자
 * @param b - 두 번째 숫자
 * @returns 두 숫자의 합
 */
function add(a: number, b: number): number {
  return a + b;
}
```

### REST API 문서화

#### GET /api/users

사용자 목록을 가져옵니다.

**요청:**
```bash
GET /api/users?page=1&limit=10
```

**응답:**
```json
{
  "users": [
    {
      "id": 1,
      "name": "홍길동",
      "email": "hong@example.com"
    }
  ],
  "total": 100,
  "page": 1,
  "limit": 10
}
```
