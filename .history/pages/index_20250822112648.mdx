# 환영합니다!

이것은 Nextra로 만든 문서 사이트입니다. 마크다운으로 작성하면 자동으로 아름다운 문서 사이트로 변환됩니다.

## 주요 기능

- 📝 **마크다운 지원** - 간단한 마크다운 문법으로 문서 작성
- 🎨 **아름다운 디자인** - 기본 제공되는 깔끔한 테마
- 🔍 **검색 기능** - 문서 내용을 빠르게 검색
- 📱 **반응형 디자인** - 모바일, 태블릿, 데스크톱 모두 지원
- 🌙 **다크 모드** - 자동으로 다크/라이트 모드 전환

## 빠른 시작

### 설치

```bash
npm install next nextra nextra-theme-docs
```

### 설정

`next.config.mjs` 파일에서 Nextra를 설정합니다:

```js
import nextra from 'nextra'

const withNextra = nextra({
  theme: 'nextra-theme-docs',
  themeConfig: './theme.config.tsx',
})

export default withNextra()
```

## 예제 코드

여기 간단한 React 컴포넌트 예제가 있습니다:

```jsx
import React from 'react'

function Welcome({ name }) {
  return <h1>안녕하세요, {name}님!</h1>
}

export default Welcome
```
