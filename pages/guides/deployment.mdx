# 배포하기

Nextra로 만든 문서 사이트를 Vercel에 배포하는 방법을 알아봅니다.

## Vercel 배포

Vercel은 Next.js를 만든 회사에서 제공하는 호스팅 서비스로, Nextra 프로젝트를 가장 쉽게 배포할 수 있습니다.

### 1단계: Git 저장소 준비

먼저 프로젝트를 Git 저장소에 푸시합니다:

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/your-username/your-repo.git
git push -u origin main
```

### 2단계: Vercel 계정 생성

1. [Vercel 웹사이트](https://vercel.com)에 접속
2. GitHub 계정으로 로그인
3. "New Project" 클릭

### 3단계: 프로젝트 가져오기

1. GitHub 저장소 목록에서 프로젝트 선택
2. "Import" 클릭
3. 빌드 설정은 자동으로 감지됨

### 4단계: 배포

"Deploy" 버튼을 클릭하면 자동으로 빌드 및 배포가 시작됩니다.

## 환경 변수 설정

필요한 경우 Vercel 대시보드에서 환경 변수를 설정할 수 있습니다:

1. 프로젝트 설정으로 이동
2. "Environment Variables" 탭 선택
3. 변수 추가

## 커스텀 도메인

Vercel은 기본적으로 `your-project.vercel.app` 도메인을 제공하지만, 커스텀 도메인도 설정할 수 있습니다:

1. 프로젝트 설정에서 "Domains" 탭 선택
2. 도메인 추가
3. DNS 설정 안내에 따라 설정

## 자동 배포

GitHub에 푸시할 때마다 자동으로 배포됩니다:

```bash
git add .
git commit -m "Update documentation"
git push
```

몇 분 내에 변경사항이 라이브 사이트에 반영됩니다!
