# 마크다운 활용법

Nextra는 표준 마크다운 문법과 함께 다양한 확장 기능을 지원합니다.

## 기본 문법

### 제목

```markdown
# 제목 1
## 제목 2
### 제목 3
```

### 텍스트 스타일

- **굵게**: `**텍스트**` → **텍스트**
- *기울임*: `*텍스트*` → *텍스트*
- ~~취소선~~: `~~텍스트~~` → ~~텍스트~~
- `인라인 코드`: `` `코드` `` → `코드`

### 목록

**순서 없는 목록:**
- 항목 1
- 항목 2
  - 중첩 항목

**순서 있는 목록:**
1. 첫 번째
2. 두 번째
3. 세 번째

## 코드 블록

### 기본 코드 블록

```javascript
function greet(name) {
  console.log(`안녕하세요, ${name}님!`);
}
```

### 라인 하이라이트

```javascript {1,3-4}
function calculate(a, b) {
  const sum = a + b;
  const diff = a - b;
  return { sum, diff };
}
```

### 파일명 표시

```typescript filename="utils/math.ts"
export function add(a: number, b: number): number {
  return a + b;
}
```

## 표

| 기능 | 설명 | 예시 |
|------|------|------|
| 제목 | 페이지 제목 | `# 제목` |
| 굵게 | 굵은 텍스트 | `**텍스트**` |
| 링크 | 하이퍼링크 | `[텍스트](URL)` |
